
# תיקון כפתור התחברות בכותרת

## הבעיה
כפתור אייקון המשתמש (👤) בכותרת הראשית של האפליקציה לא מפעיל את ההתחברות עם Google. במקום זאת, הוא מציג הודעת "התחברות תהיה זמינה בקרוב!" שהיא placeholder ישן.

## הפתרון
לחבר את כפתור אייקון המשתמש בכותרת לפונקציונליות ההתחברות הקיימת - ניווט לדף `/auth`.

## שינויים נדרשים

### קובץ: `src/components/ShoppingList.tsx`

**שינוי יחיד** - החלפת ה-onClick של כפתור המשתמש:

**לפני:**
```typescript
onClick={() => {
  toast.info(language === 'he' ? 'התחברות תהיה זמינה בקרוב!' : 'Login coming soon!');
}}
```

**אחרי:**
```typescript
onClick={() => navigate('/auth')}
```

## התנהגות צפויה אחרי התיקון
- לחיצה על אייקון המשתמש (👤) בכותרת → מעבר לדף ההתחברות (`/auth`)
- בדף ההתחברות → כפתור "התחבר עם Google" שעובד
- לאחר התחברות מוצלחת → חזרה אוטומטית לדף הראשי

## פרטים טכניים
- הקובץ `ShoppingList.tsx` כבר מייבא את `useNavigate` ומשתמש בו
- אין צורך בשינויים נוספים
- התיקון הוא שורה אחת בלבד
