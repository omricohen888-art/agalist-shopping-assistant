
# שיפורי צבע והודעת התחברות לאורחים

## סיכום
שני שינויים מבוקשים:
1. **הכהיית הצהוב** - הצבע הראשי (primary) נבלע ברקע הבהיר
2. **הודעה קופצת לאורחים** - הודעה ידידותית בכניסה לאפליקציה למשתמשים שלא מחוברים

---

## חלק 1: הכהיית הצבע הצהוב

### המצב הנוכחי
הצבע הראשי מוגדר ב-`src/index.css`:
```css
--primary: 47.9 95.8% 53.1%;  /* צהוב בהיר מדי */
```

### הפתרון
הורדת ערך ה-Lightness (בהירות) מ-53.1% ל-45% עד 48% לקבלת צהוב עמוק יותר:
```css
--primary: 45 95% 45%;  /* צהוב-זהב עמוק יותר */
```

### קובץ לשינוי
- `src/index.css` - שורה 20

---

## חלק 2: הודעה קופצת לאורחים

### ההתנהגות הרצויה
- מופיעה פעם אחת בכניסה הראשונה למסך הראשי
- רק למשתמשים שלא מחוברים (לא logged in)
- לא מופיעה שוב לאחר סגירה (שמירה ב-localStorage)
- עיצוב ידידותי ומזמין

### תוכן ההודעה
**עברית:**
> ברוכים הבאים לאגליסט!
> 
> הירשם עכשיו כדי:
> - לשמור את הרשימות שלך לאורך זמן
> - לקבל תובנות כלכליות על הקניות שלך
> - לגשת לרשימות מכל מכשיר
>
> [התחבר עכשיו] [אולי אח"כ]

**אנגלית:**
> Welcome to Agalist!
> 
> Sign up now to:
> - Save your lists permanently
> - Get financial insights on your shopping
> - Access your lists from any device
>
> [Sign In Now] [Maybe Later]

### מימוש טכני

**קובץ חדש:** `src/components/WelcomePrompt.tsx`
- קומפוננטה נפרדת לניהול ההודעה
- משתמשת ב-Dialog מ-shadcn/ui
- בודקת את מצב ההתחברות דרך useAuth
- שומרת ב-localStorage האם המשתמש ראה את ההודעה

**שינוי קיים:** `src/components/ShoppingList.tsx`
- הוספת import של WelcomePrompt
- הצבת הקומפוננטה בתוך ה-JSX

### לוגיקת הצגה
```text
אם (משתמש לא מחובר)
  וגם (לא ראה את ההודעה קודם)
  וגם (האפליקציה סיימה לטעון)
    → הצג הודעה לאחר 2 שניות
```

2. יצירת קומפוננטה `WelcomePrompt.tsx`
3. הוספת import ב-`ShoppingList.tsx`
4. הוספת הקומפוננטה ל-JSX

## פרטים טכניים

### תלויות קיימות (אין צורך בהתקנות)
- `@radix-ui/react-dialog` - כבר מותקן
- `useAuth` hook - כבר קיים
- `localStorage` - native API

### localStorage Keys
- `hasSeenWelcomePrompt` - boolean לסימון האם המשתמש ראה את ההודעה

### אינטגרציה עם AuthContext
```typescript
const { user, loading } = useAuth();
const shouldShow = !user && !loading && !hasSeenPrompt;
```

## תוצאה צפויה

- צהוב עמוק וברור יותר בכל רחבי האפליקציה
- אורחים יראו הודעה ידידותית שמזמינה אותם להירשם
- לחיצה על "התחבר עכשיו" → מעבר לדף `/auth`
- לחיצה על "אולי אח"כ" → סגירה ושמירה שלא להציג שוב
